## Recent Updates

Stay informed about the latest enhancements to Paperguide-hulku, your all-in-one AI research assistant. We release updates regularly to improve AI search, literature reviews, reference management, and more.

<Update label="2024-10-15" description="v1.3.0" tags={["feature", "improvement"]}>

## New Features

- Added Deep Research mode for advanced paper analysis and synthesis
- Introduced AI Writer enhancements with citation auto-generation
- New integration for exporting references to Zotero and Mendeley

## Improvements

- Faster AI Search results with improved relevance scoring
- Enhanced Literature Review dashboard with interactive timelines

## Bug Fixes

- Fixed occasional crashes in bulk reference imports
- Resolved sync issues across devices

</Update>

<Update label="2024-09-20" description="v1.2.0" tags={["feature", "bugfix"]}>

## New Features

- Literature Review tool now supports collaborative annotations
- AI Writer gains support for systematic review outlines
- Reference Manager adds duplicate detection and merging

## Improvements

- Optimized performance for large document libraries `{>10,000}` papers
- Better mobile responsiveness in the dashboard

## Bug Fixes

- Corrected API rate limiting errors during peak usage
- Fixed search filters not persisting across sessions

</Update>

<Update label="2024-08-10" description="v1.1.0" tags={["feature", "breaking"]}>

## New Features

- Launched AI Search with semantic understanding for complex queries
- Initial Reference Manager with PDF annotation tools
- Basic integration with academic databases like PubMed

## Breaking Changes

- Updated authentication flow to use JWT tokens instead of sessions

## Bug Fixes

- Patched security vulnerability in file uploads

</Update>
