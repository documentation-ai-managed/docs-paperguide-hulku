## Overview

In Paperguide-hulku, you can organize your documentation using Spaces and Folders.

## How To

1. Navigate to the dashboard after login.
2. Click "New Space" to create a project space.
3. Within a Space, click "Add Folder" for categories.
4. Drag and drop documents between folders.
5. Use tags for quick searching.

Organize your work to keep documentation clear and accessible.
